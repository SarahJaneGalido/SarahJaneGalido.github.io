# Sarah Jane Galido - Virtual Assistant Portfolio

A modern, responsive portfolio website for Sarah Jane Galido, showcasing services as a General Virtual Assistant and Social Media Manager.

## Features

- ✨ **Responsive Design** - Mobile-first approach, works perfectly on all devices
- 🎨 **Modern Aesthetics** - Clean, professional design with smooth animations
- 📱 **Mobile Menu** - Hamburger menu for mobile navigation
- 🔗 **Social Media Integration** - Direct links to all social platforms
- 📧 **Contact Integration** - Email and messaging capabilities
- ⚡ **Performance Optimized** - Fast loading, smooth scrolling
- 🎯 **SEO Friendly** - Proper semantic HTML structure

## Sections

1. **Hero Section** - Eye-catching introduction with call-to-action
2. **About Me** - Personal introduction and professional statement
3. **Services Offered** - 9 core services with descriptions
4. **Tools & Systems** - Tools and platforms proficiency
5. **Certificates** - Professional certifications showcase
6. **My Works** - Portfolio examples and case studies
7. **Contact** - Multiple contact methods and CTAs
8. **Footer** - Links and copyright information

## Getting Started

### Local Development

1. **Clone or navigate to the project directory**
   ```bash
   cd c:\laragon\www\sjpgalido_va_portfolio
   ```

2. **Open in a web server**
   - Using Laragon: Just access `http://sjpgalido_va_portfolio.local`
   - Or start any local server: `python -m http.server 8000`

3. **Open in browser**
   ```
   http://localhost:8000 (if using Python)
   or
   http://sjpgalido_va_portfolio.local (if using Laragon)
   ```

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #c4a574;
    --primary-dark: #a68564;
    --bg-light: #f5f1ed;
    --text-dark: #333333;
    --text-light: #666666;
}
```

### Content
Update `index.html` with:
- Personal information
- Service descriptions
- Certificate details
- Portfolio works
- Social media links

### Services
Modify the services grid in `index.html` to match your offerings. Each service card includes:
- Icon (FontAwesome)
- Title
- Description

### Tools
Add or remove tools in the tools section by modifying the tool items with appropriate icon URLs.

## Key Features Explained

### Mobile Navigation
- Hamburger menu appears on screens smaller than 768px
- Smooth toggle animation
- Auto-closes when a link is clicked

### Smooth Scrolling
- All anchor links enable smooth page scrolling
- Scroll position tracking for active section highlighting

### Animations
- Fade-in animations for cards on scroll
- Hover effects for interactive elements
- Smooth color transitions

### Responsive Breakpoints
- **Desktop**: Full layout with multi-column grids
- **Tablet** (768px and below): Adjusted layouts
- **Mobile** (480px and below): Single-column layouts
